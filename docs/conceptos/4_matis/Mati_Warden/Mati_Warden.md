# 🌀 Mati Warden (El Carcelero)

*   **Categoría:** Variante de Mati / Guardián del Consejo.
*   **Origen:** El juez y carcelero encargado de contener a las variantes inestables o rebeldes de Mati en el multiverso.
*   **Poder Principal:** *Intercambio Causal y Consecuencias de Daño.*
    *   Vincula el daño de dos objetivos; si uno recibe daño, el otro sufre las mismas consecuencias físicas inmediatas de manera invertida.
*   **Estética Visual:** Traje blindado pesado gris oscuro con runas de contención de color verde claro y un casco que oculta su rostro.
